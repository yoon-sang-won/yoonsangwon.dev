# 글 발행 체크리스트

`draft`를 해제하기 직전에 사용한다. 하나라도 확신할 수 없으면 초안으로 유지한다.

- [ ] **사실 확인:** 날짜, 버전, 인용과 외부 자료를 다시 확인했다.
- [ ] **실제 경험 여부:** 직접 경험한 내용과 추측·의견을 구분했고, 하지 않은 경험이나 성과를 쓰지 않았다.
- [ ] **비공개 정보:** 회사·고객·동료·가족 등 다른 사람의 비공개 정보가 없다.
- [ ] **코드와 이미지:** 계정 정보, 내부 주소, token, key, 쿠키와 비공개 경로가 없다.
- [ ] **제목과 본문:** 제목과 description이 실제 본문 내용과 일치한다.
- [ ] **문장:** 같은 문장과 결론을 반복하지 않았고 과장된 AI식 표현을 걷어냈다.
- [ ] **모바일:** 제목, 본문, 표·코드·이미지가 읽히며 가로 스크롤이 생기지 않는다.
- [ ] **링크:** 내부·외부 링크가 열리고 링크 문구만 읽어도 목적지를 알 수 있다.
- [ ] **이미지 alt:** 모든 의미 있는 이미지에 문맥에 맞는 alt가 있다.
- [ ] **메타데이터:** `title`, `description`, `published`, 필요 시 `updated`가 [콘텐츠 모델](content-model.md)과 맞다.
- [ ] **공개 상태:** 공개할 글만 `draft`를 해제했고 저장 후 GitHub diff를 확인했다.
- [ ] **배포:** GitHub Actions가 성공했고 공개 URL, 목록, Archive, RSS와 sitemap을 확인했다.

AI를 사용한 글은 [AI 글쓰기 운영 흐름](ai-writing-workflow.md)의 사람 검토 항목도 확인한다.
